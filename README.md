# themesv

<!-- badges: start -->

<!-- badges: end -->

The package `themesv` contains a ggplot theme theme\_sv for making
uniformly formatted viz in the SCORE network

## Installation

You can install the GitHub version of themesv with:

``` r
install_github("bmacGTPM/themesv")
```
